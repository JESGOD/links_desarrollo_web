ETIQUETAS HTML
* <section> se usa para encerrar las secciones de la pagina web, si hay un <h> se debe usar la etiqueta section para encerrar esa seccion
* <div> se usa siempre y cuando no se cumplan ninguna de las condiciones par usar una etiqueta diferente,(section, main,article etc...)
* <form> parea formularios
* <fieldset> El elemento fieldset (grupo de campos) permite organizar en grupos los campos de un formulario.
* <legend> crea un título para el fieldset es decir para el grupo de campos del formulario
